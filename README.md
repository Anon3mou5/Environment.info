# .info
It is Basically a Weather app which provide variety of weather information in your current Area or Any Other City or Street of Any City over the world.
You can even search weather information about any Any location all around the world

# Download

https://www.dropbox.com/sh/vbt34mcuri77d7s/AADQManADbANlR1GFZHiF0qea?dl=0

## The Data(Information) it provides

1)Temperature(For SURE!)

2)Humidity

3)Pressure

4)Real Temp FEEL

5)Wind Speed

# How it works

==> It determines latitude and  longitude of Current location Through either Mobile Data-net or Gps availibility.

==> Sends the obtained data to OpenWeatherAPI through AsynchHTTPClient

==> Retrives information asynchronously and loads into modules

==> Rich Lotie Animation and Transition

# Branches

==> 2 branches with 2 different UI

# Error-Handling

1)No internet-connection

2)Location not available

3)Wrong city search


#Dependency 

==>API 23 and Lottie Animations
